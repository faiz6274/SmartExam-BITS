# SmartExam - Expo Starter

This is the ready-to-run Expo starter for SmartExam.

Quick steps:
1. Replace `YOUR_BACKEND_URL` placeholders in `src/api/axios.js` and `src/screens/*` with your backend URL (including port if needed).
2. Install deps:
   ```
   npm install
   npx expo install react-native-screens react-native-safe-area-context expo-image-picker expo-image-manipulator
   ```
3. Run:
   ```
   npm start
   ```
   then open in Expo Go or an emulator.
